# Analyses

This directory is the central library for generated codebase-understanding packs.

Git policy:

- Generated analysis runs are ignored by default.
- Curated analysis packs may be explicitly unignored in the root `.gitignore`.
- Static `site/` outputs and `*-manual.json` scratch checks are not tracked because they can be regenerated with `cbu visual-pack` or `cbu render-site`.
